# Section Mockup Specifications

This document outlines the visual structure of the V2 Terminal Noir layout.

## 1. Global Navigation (System Bar)
- **Top Bar:** Fixed `sticky` status bar. `SYS.ONLINE`, `VAD://LOCAL`, Memory/CPU trackers. Toggle Mode button.
- **Header:** Contains `KOE` (Righteous font) + Japanese character.
- **Links:** `SYS.FEATURES`, `SYS.SPECS`, `SYS.FAQ`. Glitch hover effects on all.
- **Source:** Hard right `[SOURCE]` link to GitHub.

## 2. Hero Section
- **Layout:** 60/40 Split.
- **Left:** System initiation text. Massive typography (`YOUR VOICE EVERYWHERE`). Brutalist `EXECUTE DOWNLOAD` button.
- **Right:** Terminal UI block. Audio EQ visualization using flickering colored bars.
- **Divider:** Infinite scrolling Marquee tape at the bottom of the hero.

## 3. Features (System Advantages)
- **Layout:** Stacked raw-border rows.
- **Index:** Massive outline numbers (`01`, `02`) in `Zinc` that turn `Amber` on row hover.
- **Content:** Icon + `Righteous` Header. `IBM Plex Mono` description.

## 4. Specs / Workings (Hardware Integration)
- **Layout:** Half/Half grid.
- **Visual:** A raw diagrammatic representation of `CTRL+SHIFT+SPACE` floating over a giant `声` pattern.
- **Comparison Table:** Constructed entirely out of raw `<div>` borders. Highlights `KOE [V1.0]` ($0) vs `OS NATIVE` (Closed).

## 5. FAQ
- **Layout:** `<details>` blocks separated by raw zinc borders.
- **Formatting:** Questions in `Righteous` text prefixed with `[Q 01]`. Answers prefixed with `> RESPONSE:` in `Amber` monospace.

## 6. Footer Noir
- **Visual:** Massive central text: `END OF THE LINE.` with `TYPING` crossed out in `Crimson`.
- **CTA:** Floating Brutalist button with solid offset shadows `ENGAGE GITHUB`.
- **Base:** Copyright string, repository links, privacy directive link.
