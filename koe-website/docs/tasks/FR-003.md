## 📋 Objective

Create a 3-column feature grid showcasing Koe's key value propositions: Lightning Fast, Privacy First, and Completely Free.

---

## 🎯 Requirements

### Feature Cards

**Feature 1: Lightning Fast**
- Icon: `Zap` (Lucide)
- Title: "Lightning Fast"
- Description: "216x real-time transcription. It's basically instant."

**Feature 2: Privacy First**
- Icon: `Shield` (Lucide)
- Title: "Privacy First"
- Description: "Local VAD processing. Your voice never leaves your machine until you choose to send it."

**Feature 3: Completely Free**
- Icon: `Gift` (Lucide)
- Title: "Completely Free"
- Description: "No subscriptions. No limits. Just your voice, transcribed."

### Visual Design
- Glassmorphism cards with subtle borders
- Amber accent glow on hover
- Icons in amber color
- Centered text alignment

---

## ✅ Acceptance Criteria

- [ ] 3-column grid on desktop (lg breakpoint)
- [ ] 2-column on tablet (md breakpoint)
- [ ] Single column stack on mobile
- [ ] Consistent icon size (48px) and style
- [ ] Scroll-triggered fade-in animation for each card
- [ ] Card hover state with subtle lift/glow effect
- [ ] Icons use accent color (#f59e0b)
- [ ] Text has proper contrast and readability
- [ ] Cards have equal height within each row
- [ ] Section has proper padding and spacing
