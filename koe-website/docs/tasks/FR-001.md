# Task: FR-001 — Hero Section

**Feature ID:** FR-001  
**Priority:** P0 (Ship-blocking)  
**Status:** Not Started  
**Created:** 2026-03-04

---

## 📋 Objective

Create an immersive, bold hero section that immediately communicates Koe's value proposition and converts visitors into downloads.

---

## 🎯 Requirements

### Content
- **Headline:** "Your Voice, Anywhere" 
- **Subheadline:** "Lightning-fast voice dictation for Windows. Free forever."
- **Japanese Character:** Display "声" (voice) as subtle design accent
- **Primary CTA:** "Download for Windows" button
- **Secondary CTA:** "View on GitHub" link
- **Version Badge:** Display latest version number (e.g., "v1.0.0")

### Visual Elements
- **Animated Waveform:** CSS or canvas-based audio visualizer that responds to scroll
- **Dark Background:** Deep obsidian (#0a0a0f)
- **Typography:** Large, impactful headline with Geist font
- **Accent Color:** Warm amber (#f59e0b) for CTAs

---

## ✅ Acceptance Criteria

- [ ] Hero renders full viewport height on all devices
- [ ] Headline uses large typography (responsive sizing)
- [ ] Japanese character "声" displays as design accent
- [ ] Animated waveform plays smoothly (60fps)
- [ ] Primary CTA button links to GitHub release download
- [ ] Secondary CTA links to GitHub repository
- [ ] Version number dynamically fetched from GitHub API
- [ ] Loading state shown while fetching version
- [ ] Works on mobile, tablet, and desktop
- [ ] Respects `prefers-reduced-motion` setting
- [ ] LCP < 3 seconds for hero content
