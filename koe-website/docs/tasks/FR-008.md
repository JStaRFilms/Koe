**Sections:**

   - Brief statement about privacy commitment
   - "Koe is built with privacy as a core principle."

2. **What We Collect**
   - "We don't collect anything."
   - Koe is completely local application
   - No analytics, no tracking, no telemetry

3. **API Key Storage**
   - Explain local-only storage of Groq API key

   - Link to GitHub repository
   - "Verify our claims - the code is open"

6. **Contact**

### Content Template
```markdown
## Privacy Policy

### Overview
Koe is built with privacy as a core principle. We believe your voice data belongs to you and you alone.

### What We Collect
**Nothing.** Koe is a completely local application. We don't collect analytics, don't use tracking cookies, and don't have telemetry of any kind.

### API Key Storage
Your Groq API key is stored locally on your Windows machine using Electron's secure storage. It never leaves your computer except when making direct API calls to Groq's servers.

### Audio Processing
- Voice Activity Detection (VAD) runs entirely on your local machine
- Audio is only sent to Groq's transcription service when you explicitly trigger dictation
- No audio is stored on disk unless you enable the History feature
- History data is stored locally and never uploaded

### Open Source
Koe is open source software. You can verify our privacy claims by examining the source code:
https://github.com/GIGAHAT1994/whisper_alt

### Contact
If you have any privacy-related questions, please open an issue on our GitHub repository.
```
