# [FR-009] System Tray

## Labels
`MUS`, `enhancement`, `skeleton`

## User Story
As a user, I want a tray icon with Start/Stop, Settings, Quit options, so the app doesn't clutter my taskbar.

## Proposed Solution

### Overview
Create a system tray icon using Electron's `Tray` module. Right-click context menu with recording toggle, settings, and quit. Tray icon updates to show recording state.

### Implementation Flow
1. Create `Tray` with custom icon on app ready
2. Build context menu: "Start Recording" / "Stop Recording", separator, "Settings", "Quit"
3. Update menu item text based on recording state
4. Single-click on tray: toggle recording
5. Show/hide floating window from tray menu
6. Minimize to tray on window close (don't quit)

### Key Considerations
- Need idle and recording variants of tray icon
- Tray icon should be 16x16 or 32x32 PNG (Windows)
- App stays running in tray when window is closed
- Tray tooltip shows current state ("Koe — Idle" / "Koe — Recording")

## Acceptance Criteria

- [ ] System tray icon is visible
- [ ] Right-click shows context menu with Start/Stop, Settings, Quit
- [ ] Tray icon/tooltip updates based on recording state
- [ ] App minimizes to tray on window close
- [ ] Quit from tray completely exits the app
