# [FR-019] Dark/Light Theme

## Labels
`COULD`, `enhancement`, `ui`

## User Story
As a user, I want to toggle themes, so the UI fits my desktop aesthetic.

## Proposed Solution

CSS custom properties for theme switching. Dark theme is default. Light theme overrides via `.theme-light` class on body.

## Acceptance Criteria

- [x] Dark theme is default
- [x] Light theme available via toggle
- [x] Theme preference persists across restarts

## Implementation Notes

- Theme selector in settings with options: Dark, Light, System (Auto)
- Light theme CSS variables added to `src/renderer/styles/main.css` under `[data-theme="light"]`
- `src/renderer/components/theme-manager.js` handles theme switching and system preference detection
- Theme applies immediately when changed (no restart required)
- Persists via electron-store
