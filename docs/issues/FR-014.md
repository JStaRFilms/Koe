# [FR-014] History Panel

**Status:** Complete ✅
**Completed Date:** 2026-03-02

## Labels
`SHOULD`, `enhancement`, `ui`

## User Story
As a user, I want a list of recent transcriptions with copy-again, so I can re-use past dictations.

## Proposed Solution

### Overview
Scrollable panel showing recent transcriptions stored in electron-store. Each entry shows timestamp, text preview, and a copy button.

### Implementation Flow
1. Store last N transcriptions in electron-store (max 100)
2. History panel UI: scrollable list, newest first
3. Each entry: timestamp, text preview (truncated), copy button
4. Click entry to expand full text
5. Toggle history panel from floating window or tray menu

### Key Considerations
- Don't store indefinitely — cap at 100 entries, FIFO
- Show both raw and enhanced text if available
- Clear history button with confirmation
- Search/filter is a COULD feature (skip for V1)

## Acceptance Criteria

- [x] Scrollable list of recent transcriptions
- [x] Each entry has timestamp and copy button
- [x] Entries expandable to show full text
- [x] History persists across restarts (electron-store)
- [x] Maximum 100 entries with FIFO rotation
