# [FR-021] System-Wide Correction Detection Outside Koe

## Labels
`Future`, `enhancement`, `desktop`, `deferred`

## User Story
As a user, I want Koe to notice when I fix pasted text in another app, so it can learn my correction without forcing me back into Koe.

## Proposed Solution

This is intentionally deferred.

Future work should investigate accessibility-driven target tracking and text-change monitoring outside Koe-owned UI, starting with Windows and then evaluating other platforms.
