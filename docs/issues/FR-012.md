# [FR-012] Language Selection

## Labels
`SHOULD`, `enhancement`, `settings`

## User Story
As a user, I want to choose my transcription language, so I can dictate in different languages.

## Proposed Solution

### Overview
Add a language dropdown to the settings panel. Pass selected language to Groq Whisper API as the `language` parameter. Default to "auto" (auto-detect).

### Implementation Flow
1. Add dropdown to settings panel with common languages
2. Store selection in electron-store
3. Pass language code to Whisper API's `language` field
4. "Auto" option lets Whisper auto-detect the language

### Key Considerations
- Whisper supports 100+ languages but auto-detect works well for most
- Language list: auto, English, Spanish, French, German, Portuguese, Japanese, Chinese, etc.
- Specifying language can improve accuracy and reduce processing time

## Acceptance Criteria

- [ ] Language dropdown in settings panel
- [ ] Selected language passed to Whisper API
- [ ] "Auto" is the default option
- [ ] Language preference persists across restarts
