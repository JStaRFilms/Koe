# [FR-011] AI Text Enhancement

## Labels
`SHOULD`, `enhancement`, `groq-integration`

## User Story
As a user, I want raw text cleaned by Llama (filler words, punctuation, grammar), so my dictation sounds polished.

## Proposed Solution

### Overview
After Whisper transcription, optionally send raw text to Groq Llama for cleanup. Use a system prompt that instructs Llama to remove filler words, fix punctuation/grammar, and maintain the speaker's intent.

### Implementation Flow
1. After transcription, if `settings.enhanceText` is true, call Llama
2. Use Groq chat completions API with `llama-3.3-70b-versatile` or similar
3. System prompt: "Clean up this dictated text. Remove filler words (um, uh, like), fix punctuation and grammar. Keep the original meaning and tone."
4. Display enhanced text in UI with option to see/copy raw
5. Toggle in settings to enable/disable

### Key Considerations
- Enhancement adds ~500ms latency — acceptable trade-off
- Llama has separate (generous) rate limits on Groq free tier
- Enhancement can change meaning — show both raw and enhanced
- Some users may prefer raw — toggle must be easy to access

## Acceptance Criteria

- [ ] Raw text is sent to Llama for cleanup when enabled
- [ ] Enhanced text removes filler words and fixes punctuation
- [ ] Both raw and enhanced text are available
- [ ] Enhancement can be toggled on/off in settings
- [ ] Enhancement failures fall back gracefully to raw text
