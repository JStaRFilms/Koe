# [FR-018] Custom Hotkey

## Labels
`COULD`, `enhancement`, `settings`

## User Story
As a user, I want to change the hotkey, so it doesn't conflict with my other shortcuts.

## Proposed Solution

Hotkey configuration field in settings. Capture keystroke combination and store it. Re-register global shortcut with new combo.

## Acceptance Criteria

- [x] Hotkey can be changed from settings
- [x] New hotkey is registered globally
- [x] Custom hotkey persists across restarts
- [x] Conflict detection warns user

## Implementation Notes

- Hotkey input field in `src/renderer/settings-window.html`
- Interactive key capture with visual feedback (pulse animation)
- `src/main/shortcuts.js` updated with `updateHotkey()` function
- Hotkey change triggers immediate re-registration via IPC
- Error handling in `src/main/ipc.js` save settings handler
- Persists via electron-store
