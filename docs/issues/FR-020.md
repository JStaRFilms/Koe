# [FR-020] Export Transcriptions

## Labels
`COULD`, `enhancement`, `ui`

## User Story
As a user, I want to export history as .txt/.md, so I can save my dictations.

## Proposed Solution

Export button in history panel. Options: `.txt` (plain text) or `.md` (markdown with timestamps). Uses Electron's `dialog.showSaveDialog()`.

## Acceptance Criteria

- [x] Export button in history panel
- [x] Supports .txt and .md formats
- [x] File save dialog with default filename
- [x] Exported file contains timestamps and text

## Implementation Notes

- Export button added to history panel footer in `src/renderer/settings-window.html`
- IPC handler added in `src/main/ipc.js` using `dialog.showSaveDialog()`
- Export API exposed in `src/main/preload.js`
- Format: `Koe Transcriptions — [date]` header with `HH:MM - [text]` entries
- Success/error toast notifications
