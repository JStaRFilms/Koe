# [FR-020] Export Transcriptions

## Labels
`COULD`, `enhancement`, `ui`

## User Story
As a user, I want to export history as .txt/.md, so I can save my dictations.

## Proposed Solution

Export button in history panel. Options: `.txt` (plain text) or `.md` (markdown with timestamps). Uses Electron's `dialog.showSaveDialog()`.

## Acceptance Criteria

- [ ] Export button in history panel
- [ ] Supports .txt and .md formats
- [ ] File save dialog with default filename
- [ ] Exported file contains timestamps and text
