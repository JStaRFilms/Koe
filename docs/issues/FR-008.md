# [FR-008] Clipboard Integration

## Labels
`MUS`, `enhancement`, `system-integration`

## User Story
As a user, I want completed text auto-copied to clipboard, so I can paste it anywhere.

## Proposed Solution

### Overview
When a transcription (or enhanced text) is finalized, automatically copy it to the system clipboard. Show a brief toast notification confirming the copy.

### Implementation Flow
1. On transcription complete: write text to clipboard via `clipboard.writeText()`
2. Show toast/notification in floating window ("Copied to clipboard ✓")
3. Toast auto-dismisses after 2 seconds
4. Keep "Copy" button for manual re-copy of last transcription

### Key Considerations
- Copy the enhanced text if enhancement is enabled, otherwise raw text
- Don't overwrite clipboard if user cancels/discards the transcription
- Toast should be subtle — not a system notification, just an in-app indicator

## Acceptance Criteria

- [ ] Completed transcription is auto-copied to clipboard
- [ ] Toast/notification confirms the copy
- [ ] Manual copy button available for re-copying
- [ ] Enhanced text is copied when enhancement is enabled
