# [FR-017] Speed vs. Accuracy Toggle

## Labels
`COULD`, `enhancement`, `settings`

## User Story
As a user, I want to choose turbo vs. large-v3 model, so I can trade speed for accuracy.

## Proposed Solution

Toggle between `whisper-large-v3-turbo` (faster, good accuracy) and `whisper-large-v3` (slower, best accuracy). Both have the same rate limits.

## Acceptance Criteria

- [ ] Toggle between turbo and standard model
- [ ] Selection persists across restarts
- [ ] Model name shown in settings
