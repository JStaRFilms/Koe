# [FR-017] Speed vs. Accuracy Toggle

## Labels
`COULD`, `enhancement`, `settings`

## User Story
As a user, I want to choose turbo vs. large-v3 model, so I can trade speed for accuracy.

## Proposed Solution

Toggle between `whisper-large-v3-turbo` (faster, good accuracy) and `whisper-large-v3` (slower, best accuracy). Both have the same rate limits.

## Acceptance Criteria

- [x] Toggle between turbo and standard model
- [x] Selection persists across restarts
- [x] Model name shown in settings

## Implementation Notes

- Added `model` setting to `DEFAULT_SETTINGS` in `src/shared/constants.js`
- UI selector added to `src/renderer/settings-window.html` with options:
  - "Fast (Turbo) — Lower accuracy, quicker" (whisper-large-v3-turbo)
  - "Accurate (Large-v3) — Higher accuracy, slower" (whisper-large-v3)
- `src/main/services/groq.js` reads model from settings, defaults to turbo
- Persists via electron-store
