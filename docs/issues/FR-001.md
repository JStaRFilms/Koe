# [FR-001] Global Hotkey Toggle

## Labels
`MUS`, `enhancement`, `skeleton`

## User Story
As a user, I want to press Ctrl+Shift+Space to start/stop recording, so I can dictate hands-free.

## Proposed Solution

### Overview
Register a global keyboard shortcut via Electron's `globalShortcut` module. Toggle recording state on press. Communicate state changes to renderer via IPC.

### Implementation Flow
1. Register `Ctrl+Shift+Space` in main process on app ready
2. On trigger, toggle `isRecording` state
3. Send `recording:toggle` IPC to renderer
4. If starting: tell renderer to begin mic capture
5. If stopping: tell renderer to stop capture, flush remaining audio

### Technical Approach

```javascript
// src/main/shortcuts.js
import { globalShortcut } from 'electron';
import { IPC_CHANNELS } from '../shared/constants.js';

export function registerShortcuts(mainWindow) {
  globalShortcut.register('CommandOrControl+Shift+Space', () => {
    mainWindow.webContents.send(IPC_CHANNELS.RECORDING_TOGGLE);
  });
}
```

### Key Considerations
- Must work even when Koe is not focused (global shortcut)
- Should unregister shortcuts on app quit to avoid orphaned bindings
- Visual indicator in floating window must update immediately
- Consider conflict detection with other apps

## Acceptance Criteria

- [ ] Pressing Ctrl+Shift+Space toggles recording state
- [ ] Shortcut works when Koe window is not focused
- [ ] Visual indicator shows recording vs idle state
- [ ] Shortcut is unregistered on app quit
