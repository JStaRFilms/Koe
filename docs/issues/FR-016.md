# [FR-016] Custom AI Prompt

## Labels
`COULD`, `enhancement`, `settings`

## User Story
As a user, I want different text styles (formal, casual, bullet points), so dictation fits my context.

## Proposed Solution

### Overview
Preset and custom prompt options for Llama text enhancement. Presets: "Clean" (default), "Formal", "Casual", "Bullet Points". Custom prompt field for power users.

### Implementation Flow
1. Add prompt selector to settings
2. Store selected preset in electron-store
3. Use preset as system prompt prefix when calling Llama
4. Custom mode: user provides their own system prompt

## Acceptance Criteria

- [x] Preset prompt styles available (clean, formal, casual, bullets)
- [x] Custom prompt input for power users
- [x] Selected style persists across restarts

## Implementation Notes

- Implemented in `src/renderer/settings-window.html` with `selPromptStyle` select element
- Options: Clean, Professional, Casual, Concise
- Backend uses `promptStyle` from settings in `src/main/services/groq.js` enhance function
- Persists via electron-store
