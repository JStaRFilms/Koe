# [FR-015] Usage Dashboard

**Status:** Complete ✅
**Completed Date:** 2026-03-02

## Labels
`SHOULD`, `enhancement`, `ui`

## User Story
As a user, I want to see daily audio seconds used and requests remaining, so I know my limits.

## Proposed Solution

### Overview
Visual usage meter showing daily consumption against Groq free tier limits. Displays requests used/remaining, audio seconds used/remaining, and a progress bar.

### Implementation Flow
1. Read daily stats from rate limiter service
2. Display in floating window footer or expandable panel
3. Progress bars: requests (X/2000), audio seconds (X/28800), RPM (X/20)
4. Color coding: green (under 50%), yellow (50-80%), red (over 80%)
5. Reset indicator: "Resets at midnight"

### Key Considerations
- Update in real-time as requests are made
- Show both absolute numbers and percentage
- Warning animation when approaching limits

## Acceptance Criteria

- [x] Daily request count displayed (X / 2,000)
- [x] Daily audio seconds displayed (X / 28,800)
- [x] Visual progress bars with color coding
- [x] Updates in real-time
- [x] Reset time shown
